# Zone Out

A React + Vite app with an Express backend.

## Setup

- **Frontend:** `npm install` then `npm run dev`
- **Backend:** See `server/` for the API server

## Scripts

- `npm run dev` — Start Vite dev server
- `npm run build` — Production build
- `npm run preview` — Preview production build
- `npm run lint` — Run ESLint

## Environment

Create a `.env` file (see `.env.example` if present) for environment variables. Do not commit secrets.
